View/Save Column Layout as Default

This function saves the current column layout as the default for any future new tabs, but only of the same folder type.  The column layout saved includes:

  • which columns are enabled,
  • column order and
  • column width.

Setting the default columns using the Default Columns... option, accomplishes nearly the same thing, but uses the current default column widths rather than the current column widths.
