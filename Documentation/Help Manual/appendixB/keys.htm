Appendix B/Accelerator, Modifier keys

Following is a list of Accelerator keys, etc. which are active in Explorer++.  Some keys are also listed which are modifiers to mouse actions (ie. drag-and-drop).

Alphanumeric Keys eg. A, B, 1, 2
Function Keys eg. F1, F2
Page Navigation Keys eg. Home, PageUp
Cursor Keys Left, Right, Up
Editing Keys Enter, Backspace, Tab
Miscellaneous Keys Apps, [, ]
Mouse Keys, etc.  

 

Alphanumeric Keys

  MODIFIER KEYS  
KEY +SHIFT +CTRL +ALT FUNCTION
A   C   Select All
B   C   Organize Bookmarks
C   C   Copy
C S C   Copy to Folder
D   C   Bookmark This Tab
D S C   Wildcard Deselect
D     A Select Address Bar
E   C   Duplicate Tab
F   C   Search
F S C   Set Filter
H   C   Show Hidden Files
I   C   Invert Selection
L   C   Select Address Bar
M S C   Move To Folder
N   C   New Folder
N S C   Select None
P S C   Copy Folder Path (to clipboard)
R   C   Refresh
S S C   Wildcard Select
T   C   New Tab
V   C   Paste
V S C   Paste Shortcut
W   C   Close Tab
X   C   Cut
Z   C   Undo
1..8   C   Select Tabs 1..8
9   C   Select Last Tab
